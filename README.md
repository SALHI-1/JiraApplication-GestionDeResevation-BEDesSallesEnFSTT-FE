# Client Frontend

This is the frontend client for the Scrum Management Project (LSI3).

## Technologies Used

- **React**: UI Library (v19)
- **Vite**: Build tool and bundler
- **TypeScript**: Static typing
- **React Router**: Navigation
- **Axios**: HTTP Client

## Installation

1. Navigate to the frontend directory:
   ```bash
   cd frontend
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Ensure environment variables are set (check `.env` or `.env.local` if applicable).

## Usage

### Development Server
To start the development server:
```bash
npm run dev
```

### Build
To build the application for production:
```bash
npm run build
```

### Preview
To preview the production build locally:
```bash
npm run preview
```

### Linting
To run the linter:
```bash
npm run lint
```
